# AI-Powered CI/CD Log Analyzer

A small DevOps project that demonstrates how GitHub Actions can be used to automate build and test workflows, while integrating AI to help analyze pipeline failures and improve troubleshooting.

## Project Goal
This project shows how AI can support DevOps and developer workflows by making CI/CD failures easier to understand.

## Tools Used
- GitHub Actions
- Python
- Docker
- AI API integration (later step)

## Planned Workflow
1. Developer pushes code to GitHub
2. GitHub Actions triggers automatically
3. Application build and test steps run
4. If a failure happens, logs are captured
5. AI analyzes the failure and provides a readable explanation
